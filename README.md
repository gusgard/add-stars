# add-stars

Add stars to all the repos in GitHub used in package.json.

## Install

```bash
$ npm i -g add-stars
```

## Setup

```bash
$ add-stars
```

#### Add authentication information for GitHub

![auth](img/auth.png)

#### Add stars to all the repos

![add-stars](img/add-stars.png)

#### Reset authentication
```bash
$ add-stars -a
```

#### Star this repo 👍
```bash
$ add-stars -s
```

#### Help
```bash
$ add-stars -h
```
