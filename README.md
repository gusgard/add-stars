# add-stars

Add stars to all the repos in GitHub used in package.json.

[![NPM](https://nodei.co/npm/add-stars.png?downloads=true)](https://nodei.co/npm/add-stars/)

## Install

```bash
$ npm i -g add-stars
```

## Setup

```bash
$ add-stars
```
